# LK Photo Day

One-off editorial landing page for a Photo Day event in Warsaw.

## Development

```bash
npm install
npm run dev
```

## Environment

Copy `.env.example` to `.env.local` and provide:

- `RESEND_API_KEY`
- `BOOKING_EMAIL_TO`
- `TELEGRAM_BOT_TOKEN`
- `TELEGRAM_CHAT_ID`
